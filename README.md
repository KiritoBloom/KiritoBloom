### What I've built

**[Campus Portal API](https://moe-portfolio.vercel.app)** &nbsp;·&nbsp; `Python` `Flask` `Redis` `Next.js` `GitHub Actions`

My university's portal was a mess — slow, unreliable, impossible to work with. So I reverse-engineered the whole thing: auth, schedules, grades, attendance, exam seating, CMS. Rebuilt it as a clean API with Redis caching at **10ms response times**, wrapped it in an Android app, and shipped it to the **Google Play Store**. 300+ students use it every day.

That's not a side project. That's infrastructure.

<br/>

**[Headless E-Commerce Platform](https://moe-portfolio.vercel.app)** &nbsp;·&nbsp; `Next.js` `TypeScript` `MongoDB` `REST APIs`

Full headless backend designed for multi-storefront support from day one. Admin dashboard for inventory, pricing, and analytics. Real-time MongoDB sync. Decoupled so the frontend is completely swappable.
